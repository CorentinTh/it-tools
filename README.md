- [ ] QR code generator

You have an idea of a tool? Submit a feature request!

## Project setup
Install dependencies by running the following command:
```shell
npm install
```

Then compiles and hot-reloads for development:
```shell
npm run serve
```

And to lint and fixe files, run:
```shell
npm run lint
```
